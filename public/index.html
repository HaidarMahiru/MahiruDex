Tentang MahiruDex

MahiruDex adalah aplikasi client pembaca manga web custom yang dibuat secara khusus agar bersih, cepat, dan bebas dari iklan mengganggu.

Layanan kami sepenuhnya menggunakan MangaDex resmi untuk memuat ribuan manga, chapter, dan detail komik secara real-time langsung dari server MangaDex.

Bebas Iklan & Proxy Gambar

Semua gambar manga di-proxy langsung melalui server kami untuk menghindari masalah CORS, hotlinking, atau loading lambat dari jaringan CDN MangaDex.

Dokumentasi API Internal

MahiruDex menyediakan endpoint API mandiri bebas CORS yang siap pakai untuk pencarian manga, filter genre, detail, bab, relasi sekuel, dan proxy gambar/iframe.
